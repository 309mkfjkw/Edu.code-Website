Loop in C Language:




In C programming, a loop is a control structure used to
repeat a block of code multiple times, based on a specific condition. Loops make it easier to perform repetitive tasks
without writing the same code repeatedly.


Type of Loop Description Syntax Example Flowchart
For Loop Used when the number of iterations is known beforehand. c
for (int i = 0; i < n; i++)
{
// Code to execute if condition is true
}
 Start → Condition → Code → Update → Back to Condition
While Loop Used when the condition needs to be checked before each iteration. c
Start
while (condition)
{ /* code */ }		 Start → Condition → Code → Back to Condition
Do-While Loop Executes the code at least once, then checks the condition. c
do
{ /* code */ }
while (condition);
}		 Start → Code → Condition → Back to Code (if true)
