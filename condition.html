Conditional statements in C Language:




Conditional statements are programming constructs that allow you to make decisions in your code based on whether a
specific condition is true or false. They enable the program to execute certain blocks of code only when the specified condition is met.





type Description Structured
if Statement Executes a block of code if a condition is true. c
if (condition) {
// Code to execute if condition is true
}
if-else Statement Executes one block of code if the condition is true, and another if the condition is false. c
if (condition) {
// Code if true
} else {
// Code if false
}
else if Ladder Allows checking multiple conditions in sequence. c
if (condition1) {
// Code if condition1 is true
} else if (condition2) {
// Code if condition2 is true
} else {
// Code if none
}
Nested if An if statement inside another if statement for multi-level conditions. c
if (condition1) {
if (condition2) {
// Code if both are true
}
}
Switch Statement Evaluates a variable against multiple values (cases) and executes the matching case. c
switch (variable) {
case value1:
// Code
break;
case value2:
// Code
break;
default:
// Code
}
